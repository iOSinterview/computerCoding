# 移除元素

---

将nums中值为`val`的元素删除掉，时间复杂度为`O(n)`，空间复杂度为`O(1)`

```Go
func removeElement(nums []int, val int) int {
    /*每个不等于val的值，向前移动前面count（前面val值的个数）*/
    n := len(nums)
    count := 0 //统计当前值前面val的个数
    for  i := 0; i < n; i ++{
        if nums[i] == val{
            count ++
        }else{
            nums[i - count] = nums[i]
        }
    }
    return n - count
}
```
