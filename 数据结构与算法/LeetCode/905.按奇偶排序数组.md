

# [905. 按奇偶排序数组](https://leetcode.cn/problems/sort-array-by-parity/)

给你一个整数数组 `nums`，将 `nums` 中的的所有偶数元素移动到数组的前面，后跟所有奇数元素。

返回满足此条件的 **任一数组** 作为答案。

**示例 1：**

```
输入：nums = [3,1,2,4]
输出：[2,4,3,1]
解释：[4,2,3,1]、[2,4,1,3] 和 [4,2,1,3] 也会被视作正确答案。
```

**示例 2：**

```
输入：nums = [0]
输出：[0]
```

**提示：**

- `1 <= nums.length <= 5000`
- `0 <= nums[i] <= 5000`

题解：

方法一：直接用一个新数组

方法二：采用双指针，左右两边，左边找到第一个奇数，接着右边找到第一个偶数，进行交换。

```go
// 将一个数组偶数放在前面，奇数放在后面（双指针）
func f(a []int) {
	n := len(a)
	i, j := 0, n-1
	for i < j {
		for a[i]%2 == 0 {
			i++
		}
		for a[j]%2 == 1 {
			j--
		}
        // 这里，要保证交换前i<j
		if i >= j {
			break
		}
		a[i], a[j] = a[j], a[i]
	}
	fmt.Println(a)
}
```
